Description
    This program implements a String reversal in Java by first converting the String to a character
    array. The first and last character are swapped, then the second and second-to-last, until the
    center is reached.  

Highlights
    -String to char[] conversion - both ways
    -Algorithm design

To Run
    StringReverse.java can run in any java environment.  Dowmload StringReverse.java to your
    machine, compile and run in your preferred java environment.
