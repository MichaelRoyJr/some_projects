Description
    This program counts the number of words in a file.  A prompt requests a filename.  If the file
    exists, the number of words it contains is counted and displayed.  For the purposes of this 
    program, a word is a string of characters separated by whitespace from another word.  If the 
    file does not exist, an exception is thrown.

Highlights
    -input methods
    -exception handling

To Run
    WordCounter.java can run in any java environment.  Dowmload WordCounter.java to your machine, 
    compile and run in your preferred java environment.
